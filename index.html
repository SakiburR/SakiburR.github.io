Bootcamp 2020: Speedy Snake Game

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, variables, classes, for loops, if statements, soundfiles, and randomness. I created a 2D Snake Game in which a 'snake' grows larger by eating 'food'. Every time the food is consumed by the snake, it grows larger, and the food respawns at a random location in the area. As the snake consumes its food one-by-one, a 'Score' is added for every piece of food it consumes, and this is displayed in the upper left corner of the sketch.

Check out the code on Github: https://github.com/SakiburR/adbt155

Naruto Energy Run - Game for IN1007 Java Coursework

This is my second programming project, where I made a Game using Java. My game is called Naruto Energy Run, and this game/project took roughly 2-3 months to make. The main objective of the game is for Naruto to collect his Chakra balls and reach the energy goal in order to advance to the next level. Once he gets the required energy, naruto can use the yellow portal to go to the next level. Along Naruto's Journey he faces multiple enemy's such as Madara, Sasuke, Kisame and Itachi. If naruto collides with the enemies he gets destroyed and loses the game. However, with Naruto's shuriken ability, he can launch shuriken at his enemies and destroy them with ease. Naruto is also faced with the red portal which sends him back to his spawnpoint. Whilst making the game I have faced many challenges and I have learnt alot. This includes problems I faced when making the save and load feature, making a throwable shuriken, and lots more. Comparing my first programming project with this one, I can tell how the module IN1007 Programming in Java has helped improve my programming skills massively.
